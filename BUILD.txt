==Linux==
mkdir build
cd build
cmake -DHSPLASMA_LIBRARY=/path/to/libHSPlasma.so -DHSPLASMA_INCLUDE=/path/to/hsplasma/core ../
make

==Windows Visual Studio==
mkdir build
cd build
cmake -G"VISUAL_STUDIO_VERSION" -DHSPLASMA_LIBRARY=/path/to/HSPlasma.dll -DHSPLASMA_IMPLIB=/path/to/HSPlasma.lib -DHSPLASMA_INCLUDE=/path/to/hsplasma/core ../

VISUAL_STUDIO_VERSION is one of:
Visual Studio 7 .NET 2003
Visual Studio 8 2005
Visual Studio 8 2005 Win64
Visual Studio 9 2008
Visual Studio 9 2008 Win64

You can now open the generated visual studio project.

==Windows MinGW==
mkdir build
cd build
cmake -G"MinGW Makefiles" -DHSPLASMA_LIBRARY=/path/to/HSPlasma.dll -DHSPLASMA_IMPLIB=/path/to/HSPlasma.lib -DHSPLASMA_INCLUDE=/path/to/hsplasma/core ../
mingw32-make
