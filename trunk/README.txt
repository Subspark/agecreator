Main points:
* Create a new age with file->new.
* Create a new layer with the 'new folder' button
* Create a new object with the '+' button
* load/save are not implemented yet
* NO SUM FILE IS CREATED ON EXPORT, you must use PlasmaSum to create it manually
* Currently, it's not possible to re-position any object. Everything is created at [0,0,0].
* To active 3d preview controls, click the preview window
 * To move forward/back, use up/down arrow keys
 * To strafe left/right, use left/right arrow keys
 * To fly up/down, use PgUp/PgDown keys
 * Hold middle button for mouse look
